# Otto DIY Crane Robot

## Descrição

Este projeto apresenta a modelagem 3D de um robô **Otto DIY** adaptado com uma extensão de **guindaste acoplada à cabeça**, desenvolvida utilizando o **Fusion 360**. O objetivo do projeto é demonstrar a integração entre robótica educacional, modelagem 3D e automação, permitindo que o robô realize movimentos de levantamento e transporte de pequenos objetos.

O projeto foi desenvolvido como atividade acadêmica, envolvendo desde a concepção mecânica até a simulação e documentação do sistema.

## Objetivos do Projeto

* Adaptar o robô Otto DIY para atuar como um pequeno guindaste móvel.
* Desenvolver uma estrutura mecânica compatível com impressão 3D.
* Simular o comportamento do sistema antes da montagem física.
* Representar a lógica de funcionamento através de Redes de Petri Coloridas.
* Integrar hardware e software em uma plataforma educacional de baixo custo.

## Funcionalidades

* Movimentação básica do Otto DIY.
* Sistema de elevação utilizando guindaste.
* Simulação virtual do funcionamento.
* Controle através de servomotores.
* Modelagem e validação da lógica por Rede de Petri.
* Estrutura totalmente modelada em CAD.

---

## Fluxo de Funcionamento

1. Inicialização do robô.
2. Posicionamento para coleta do objeto.
3. Acionamento do mecanismo de elevação.
4. Transporte do objeto.
5. Posicionamento final.
6. Liberação da carga.
7. Retorno ao estado inicial.

A lógica de controle também foi representada através de uma Rede de Petri Colorida, permitindo a análise formal dos estados do sistema e do tratamento de falhas.
