# 🤖 Otto DIY – Relatório de Montagem e Calibragem

## 📌 Descrição do Projeto

Este repositório contém o relatório completo do projeto Otto DIY, apresentando o processo de montagem, configuração, programação e calibragem do robô. O objetivo do projeto foi aplicar conhecimentos de robótica, eletrônica e programação de forma prática e organizada.

Durante o desenvolvimento foram realizados testes de funcionamento, ajustes mecânicos e calibragem dos servomotores para garantir melhor estabilidade e movimentação do robô.

Além disso, toda a documentação foi produzida utilizando ferramentas voltadas para desenvolvimento e documentação técnica.

---

## 🎯 Objetivos

* Realizar a montagem do robô Otto DIY
* Desenvolver conhecimentos em eletrônica e automação
* Aplicar programação utilizando Arduino
* Documentar o projeto em LaTeX
* Organizar os arquivos utilizando GitHub

## ⚙️ Etapas do Projeto

### 🔹 Montagem

Montagem física da estrutura do Otto DIY e instalação dos componentes eletrônicos.

### 🔹 Ligações Elétricas

Conexão dos servomotores, sensor ultrassônico e demais componentes ao Arduino.

### 🔹 Programação

Desenvolvimento e envio do código responsável pelos movimentos e funções do robô utilizando Arduino IDE.

### 🔹 Calibragem

Ajuste dos servomotores para alinhamento correto das pernas e estabilidade dos movimentos.

### 🔹 Documentação

Criação do relatório técnico utilizando LaTeX no Overleaf e no VS Code.
